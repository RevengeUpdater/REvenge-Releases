REvenge Installer - Инструкция по сборке

ТРЕБОВАНИЯ
- Visual Studio 2022 Community (бесплатно)
- Windows 10 или 11
- C++ компонент для разработки

БЫСТРЫЙ СТАРТ

1. Открой файл REvenge_Installer.sln в Visual Studio
2. Нажми Ctrl+Shift+B или Build > Build Solution
3. Готовый exe будет в папке x64\Release\

ДЕТАЛИ СБОРКИ

Visual Studio автоматически:
- Скомпилирует исходный код (main.cpp)
- Обработает ресурсы (app.ico, logo_data.h)
- Слинкует все библиотеки
- Встроит CRT статически (/MT флаг)
- Создаст один .exe файл без зависимостей

ПОСЛЕ СБОРКИ

Файл REvenge_Installer.exe готов к использованию.
Он работает на любой Windows 10 или 11 без установки дополнительного ПО.

ЕСЛИ ОШИБКИ

Ошибка с тулсетом:
- Правой кнопкой на проект > Properties
- Найди "Platform Toolset"
- Выбери доступный вариант (v143, v142 и т.д.)

Ошибка с SDK:
- Tools > Get Tools and Features
- Установи "Desktop development with C++"

КОНФИГУРАЦИЯ

Release - для финального exe (маленький, быстрый)
Debug - для тестирования (большой, с отладкой)

Переключи в верхней панели перед сборкой.
