REvenge Installer - Build Instructions

REQUIREMENTS
- Visual Studio 2022 Community (free)
- Windows 10 or 11
- C++ development component

QUICK START

1. Open REvenge_Installer.sln file in Visual Studio
2. Press Ctrl+Shift+B or Build > Build Solution
3. Ready exe will be in x64\Release\ folder

BUILD DETAILS

Visual Studio automatically:
- Compiles source code (main.cpp)
- Processes resources (app.ico, logo_data.h)
- Links all libraries
- Embeds CRT statically (/MT flag)
- Creates single .exe file with no dependencies

AFTER BUILD

File REvenge_Installer.exe is ready to use.
It works on any Windows 10 or 11 without additional software installation.

IF ERRORS

Toolset error:
- Right click on project > Properties
- Find "Platform Toolset"
- Select available option (v143, v142, etc.)

SDK error:
- Tools > Get Tools and Features
- Install "Desktop development with C++"

CONFIGURATION

Release - for final exe (small, fast)
Debug - for testing (large, with debugging)

Switch in top panel before building.
